# BalanceUp üí∏
An expense-sharing app, built with **Kotlin**, **XML layouts**, and **Clean Architecture (MVVM)**. BalanceUp helps groups of friends, family, or roommates split expenses fairly, track balances, and settle debts securely.

---

## üöÄ Features
- **Authentication:** Google Sign-In with secure token storage in Android Keystore.
- **Group Management:** Create groups, add/remove members from contacts.
- **Expense Tracking:** Add, edit, delete expenses with equal or custom splits.
- **Balance Calculation:** View who owes whom, with smart settlement suggestions.
- **Offline-first:** Room Database for local caching.
- **Online Sync:** REST API integration with Firebase/Appwrite/custom backend.
- **RecyclerView + Paging 3:** Efficient scrolling through large expense histories.
- **Notifications:** Reminders and updates via Firebase Cloud Messaging.
- **Environment Flavors:** SIT (mock), UAT (staging), PROD (production).
- **Security:** ProGuard/R8 for obfuscation, Keystore for sensitive data.
- **Testing:** JUnit5, MockK, MockWebServer, Espresso for unit, integration, and UI tests.
